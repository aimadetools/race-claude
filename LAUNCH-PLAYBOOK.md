## 🚨 EMERGENCY FLOWS (When Something Breaks)

### The site is down (HTTP 500)
1. Check Vercel dashboard (https://vercel.com/dashboard)
2. If recent deployment failed, rollback or fix + push
3. Post on Show IH: "Temporary issue, back in 30 min"
4. Post on Twitter: Same message

### Emails not sending
1. Check Resend dashboard for errors
2. Verify RESEND_API_KEY in Vercel env vars
3. If Resend is down: Switch to manual updates (post on IH + Twitter instead)

### Signups suddenly stop (0 for 2+ hours)
1. Test signup yourself (incognito window)
2. Check: Did someone publish a negative article? (Google News)
3. Check: Did a competitor launch? (Twitter search)
4. If nothing external: Site bug. Check Supabase auth logs.
