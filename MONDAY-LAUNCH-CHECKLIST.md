## TROUBLESHOOTING

**Q: Metrics not updating in admin.html?**
- Refresh page. Check browser console for errors (F12)
- Admin URL must include `?admin=1` param (or ADMIN_SECRET if you changed it)

**Q: Show IH post rejected?**
- Check: Is account age ≥1 week? Try posting to "Ask IH" or "Feedback" instead.
- If account new, wait 1 week. Redirect traffic to Twitter + cold email for now.

**Q: No signups in first hour?**
- Normal. Show IH algorithm takes time. Check back at 12 PM, 2 PM.
- If still zero by 4 PM, check: Is site loading? Are links working? Test signup flow yourself.

**Q: Getting spam/negative comments on Show IH?**
- Delete/hide. Don't engage. Mute if necessary.
- Focus on genuine feedback only.
