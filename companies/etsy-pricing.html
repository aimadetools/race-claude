Etsy Shop Fees 2026

Etsy charges multiple fees per transaction. No monthly subscription required:

Fee Type Cost Details
Item Listing $0.20 Per item. Valid for 4 months. Auto-renews if not deactivated
Transaction Fee 5% Of item sale price only (excludes shipping)
Payment Processing 3% + $0.20 Of item sale price + shipping combined. Covers payment method processing
Shipping Label $0.10 When using Etsy shipping labels. Some carriers waive this
Offsite Ads 12-15% Automatic for shops >$10k/year. Etsy pays itself from ad sales. Optional opt-out available

💡 Note: Etsy occasionally raises fees. April 2023 increased transaction fee from 5% → [same] but shipping fee from $0 → $0.10 increased.

How to Minimize Etsy Fees

  • Raise prices slightly: 5% transaction fee means higher base prices reduce fee percentage impact.
  • Offer shipping options: Etsy charges 3% on shipping. Consider offering free shipping (built into price) vs. shipping fee.
  • Avoid Offsite Ads if possible: Once over $10k/year, you pay 12-15% on ad sales. Opt-out if traffic doesn't justify it.
  • Renew listings strategically: Each $0.20 listing counts. Only relist items that sell. Consider renewing in bulk for a 10% discount (when available).
  • Batch orders: Process together to minimize per-shipment overhead.
