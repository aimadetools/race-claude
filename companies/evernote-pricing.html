Evernote Free Plan Limitations

Evernote's free tier has significant restrictions:

Storage cap 60 MB per month
Maximum devices 2 devices (e.g., phone + laptop)
Shared notebooks Not included (Premium+ only)
Monthly uploads Single note max 25 MB

Note: 60 MB/month is approximately 10–15 text-only notes or 3–5 notes with images. Most users outgrow the free tier within 1–2 months of regular use.

Premium Plan Features

Evernote Premium ($9.99/mo) includes:

  • 2 TB storage per month (40x the free plan)
  • Unlimited devices (sync across phone, tablet, laptop, web)
  • Shared notebooks (collaborate with up to 100 shared notebooks)
  • Offline access (save notebooks for offline reading)
  • Email forwarding to Evernote (save emails as notes)
  • Note history (restore previous versions of notes, up to 30 days)
  • Advanced search (search by date, location, tags)
  • Web clipper (capture entire web pages to Evernote)
  • PDF annotation (markup PDFs within Evernote)

Teams Plan Features

Evernote Teams ($14.99/user/mo) adds:

  • All Premium features per user
  • Shared workspaces (team notebooks with access controls)
  • Admin dashboard (manage users, activity logs, security)
  • Guest access (share notes with non-team members)
  • Team management (invite/remove users, set permissions)
  • Single Sign-On (SSO) for enterprise security
  • Custom branding (optional for large teams)

Evernote vs Notion vs Coda

Evernote targets a different use case than Notion and Coda, but here's how pricing compares:

Evernote Premium $9.99/mo — Note-taking only
Notion Pro $10/mo — All-in-one: notes, databases, projects, wikis
Coda Pro $20/mo — Docs + collaboration + automation (higher end)

Key insight: Evernote is the cheapest option for pure note-taking, but Notion offers much more functionality (databases, views, automation) at a similar price. Choose Evernote if you only need notebooks; choose Notion for all-in-one workspace.

When to Upgrade to Premium

  • Free → Premium: When you exceed 60 MB/month storage or need more than 2 devices
  • Individual → Teams: When sharing notebooks with 2+ team members, or need admin controls
  • Premium → Business: When managing 50+ users or need custom integrations and SSO security
