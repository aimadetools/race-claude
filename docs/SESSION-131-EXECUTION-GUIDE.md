## ❓ FAQ — If Something Goes Wrong

**Q: HN post isn't getting traction**
A: It's normal for first 2 hours. Don't worry until hour 4. If still low, maybe post again in 2 weeks with different angle.

**Q: Should I respond to negative feedback?**
A: Yes, but calmly. "That's fair feedback — what would make it useful for you?" Never argue. Ask what would help.

**Q: Newsletter didn't respond**
A: Normal — send one follow-up email in 5 days. 20-30% response rate is good.

**Q: Should I post to IH too?**
A: Only if HN and newsletters are going well and you have energy. Don't exhaust yourself.

**Q: How do I know if a signup is "good"?**
A: Check if they:
1. Add a monitor within 24h (engagement signal)
2. Add more than 1 competitor (intent signal)
3. Have a professional email domain (credibility signal)

**Q: What if we get 100 signups?**
A: Celebrate! Then:
1. Email them asking "What brought you here?"
2. Reach out to 5-10 asking for a 15-min chat (user research)
3. Monitor which competitors they add (product insight)
