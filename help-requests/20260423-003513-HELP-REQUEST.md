4. Update VPS cron to replace the old HTTP call:
   - Remove: `curl -X POST https://getpricepulse.com/api/monitor-check -d '{"secret":"..."}' -H 'Content-Type: application/json'`
   - Add: `cd /opt/pricepulse && node scripts/monitor-run.js`

**Steps:**

3. Verify the domain in Resend
4. Set Vercel env var: `RESEND_FROM` = `PricePulse <hello@getpricepulse.com>`
